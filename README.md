## 🚀 Deployment

This website is deployed using GitHub Pages. When you commit and push to the main branch, the website automatically updates.

### Setting up GitHub Pages:

1. Go to your repository on GitHub
2. Click on "Settings"
3. Scroll down to the "GitHub Pages" section
4. Select the branch you want to deploy (usually "main")
5. Select the folder (usually "/ (root)")
6. Click "Save"
7. Your site will be published at https://[your-github-username].github.io/[repository-name]/

## 🔄 Development

To run this website locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/ayushpatil0810/portfolio-website.git
   ```

2. Navigate into the project directory:
   ```bash
   cd portfolio-website
   ```

3. Open `index.html` in your browser, or use a simple HTTP server:
   ```bash
   # Using Python 3
   python -m http.server
   ```

4. The site should now be running at `http://localhost:8000`
